# cmd

## 실행하기

- 관리자 권한으로 실행
- 계정에 암호가 부여되어야 공유가능
- 블록잡고 shift + enter 하면 블록 잡힌 부분 복사됨



## 코드

- c 드라이브 프롬프트로 이동

  ```bash
  $ cd /
  ```

* cmd 화면창 청소

  ``` bash
  $ cls
  ```

* 아이피 정보 확인

  ```bash
  $ ipconfig /all
  ```

* 네트워크 연결 오픈 (ip세팅하는 화면)

  ``` bash
  $ ncpa.cpl
  ```

* 공유 폴더 접속(예시 공백 주의)

  ``` bash
  $ net use v : \\(ip주소)\share
  ```
